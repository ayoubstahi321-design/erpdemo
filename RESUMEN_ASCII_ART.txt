LIBRERÍA PRINCIPAL (pdfGenerator.ts):

  ┌─ generateProfessionalPDF(options) ─────────────────────────────┐
  │  • Función principal que genera PDFs nativos                  │
  │  • Retorna: Blob (PDF real)                                   │
  │  • Tiempo: 150-200ms                                          │
  │  • Soporta: A4 y TICKET                                       │
  └──────────────────────────────────────────────────────────────┘

  ┌─ downloadPDF(blob, filename) ─────────────────────────────────┐
  │  • Descarga PDF al dispositivo                                │
  │  • Maneja ObjectURL correctamente                             │
  │  • Limpia recursos automáticamente                            │
  │  • Compatible: Todos los navegadores                          │
  └──────────────────────────────────────────────────────────────┘

  ┌─ sharePDF(blob, filename, title) ─────────────────────────────┐
  │  • Comparte PDF con Web Share API                             │
  │  • Plataformas: Email, WhatsApp, Drive, etc                   │
  │  • Compatible: iOS 13+, Android 5+                            │
  │  • Fallback: Alert si no soportado                            │
  └──────────────────────────────────────────────────────────────┘
